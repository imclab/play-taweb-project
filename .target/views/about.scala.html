Sources

This website is dedicated to fetch data from external sources.

  • DBpedia (dbpedia.org) is used for parsing cities' information in RDF/XML format from his SPARQL EndPoint.
  • Open Weather Map (openweathermap.org) is used for parsing weather forecast information in JSON format.
  • Flickr (flickr.com) is used for getting photos in XML format.
