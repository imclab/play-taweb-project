About 4travelers Ontology
v1.0

RDF Travel Ontology

Vocabulary for expressing traveling and destinations using the Resource Description Framework.

Namespace

The URI for this vocabulary is http://4travelers.org/

When abbreviating terms the suggested prefix is trvl

Each class or property in the vocabulary has a URI constructed by appending a term name to the vocabulary URI. For example: http://4travelers.org/Destination
