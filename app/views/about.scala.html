Description
4Travelers v1.0

This website is dedicated to fetch/handle data from external sources in order to produce semantic data in RDF format with the addition of users' recommendations.

External sources

DBpedia (dbpedia.org) is used for parsing cities' information in RDF/XML format from his SPARQL EndPoint.

Open Weather Map (openweathermap.org) is used for parsing weather forecast information in JSON format.

Flickr (www.flickr.com) is used for getting photos in XML format.
